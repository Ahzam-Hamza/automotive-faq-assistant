🚗 Built a Source-Grounded Automotive FAQ Assistant using GenAI concepts — local & reliable

Ever felt lost searching for simple answers inside a vehicle manual?

I built a GenAI-powered Automotive FAQ Assistant that lets users ask natural-language questions about a car and get accurate answers grounded strictly in the vehicle manual.

🔍 What it does:

• Answers maintenance-related questions (engine oil, tire pressure, battery, warranty)
• Uses the car manual as the only source of truth
• Avoids AI hallucinations by clearly stating when information is unavailable
• Designed with safety-critical automotive use cases in mind

🧠 Key idea:
Source-aware answering inspired by Retrieval-Augmented Generation (RAG) —
the system first identifies the relevant manual section and then generates answers only from that content.

🛠 Built with:
Python, structured manual data, source-grounded logic, and clear documentation
